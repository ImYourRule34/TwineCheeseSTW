Basic AHK script.

Idea inspiration from: FaaX Ultimate on YT
https://www.youtube.com/watch?v=99VBw-xHP00&ab_channel=FaaXUltimate

To change configuration, compile using AHK. 
Uses AHK v2

Just runs these keypress events in order:
f e e f x

Change interact and other menu navigation
items as needed in each of the keypress events.

My local copy of its .exe is stored in this last commit.

To use, go to position with appropriate construction as
demonstrated in the video. Only works on Twine Peaks.

While in a position to interact with defender pad, press [`].
